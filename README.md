### Pour mettre flutter dans le PATH
Mettre dans le bashrc
```
echo 'export PATH="$PATH:$HOME/Bureau/flutter/bin"' >> ~/.bashrc
export PATH="$PATH:`pwd`/flutter/bin"
source ~/.bashrc
```
## Lancement my_flutter_hello_app
```

flutter create .
```
